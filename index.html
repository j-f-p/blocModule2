A Portfolio of Web and Game Projects

by Fernando Padilla

compatible with Chrome 73, Firefox 66 and Edge 42

Subject Notes

This is a web server application for collaboratively compiling notes about a subject. The app is developed with Python, Flask and PostgreSQL. It can authenticate any user with a Google account.

compatible with Chrome 70, Firefox 63 and Edge 42

Matching Game

This is a one player game about revealing cards by matching. The game is built with HTML, CSS and JavaScript.

compatible with Chrome 70, Firefox 63 and Edge 42

Image Clicker Album

This is an album of images. Each default image can be swapped for an image of choice, by specifying a new URL. Their labels may also be edited. The app is built with HTML, CSS and JavaScript.

demonstration upon request

Bloccit

This is a discussion site built with Ruby on Rails and PostgreSQL. It includes authentication, authorization and voting.

demonstration upon request

Connect 4

This is the Connect 4 game completed with Java. It contains an algorithm for determining connect 4 game moves.

demonstration upon request

Copter Run Demo

This is a 2D sideview helicopter game. It is written in Python with the Pygame library.
