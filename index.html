A Portfolio of Exhibition Projects

by J. F. Padilla

Browsers compatible with video demo page include recent versions of Chrome, Edge and Firefox.

Disk Simulation

This is a 2D simulation of binary collisions among disks moving along straight paths. Separate threads concurrently process the animation and the physics. This project is written in C++. It employs the SDL library for rendering graphics.

Linux Monitor

This is a Linux system monitor. Its output includes memory and CPU usage of the system's most intense software processes. This project is written in C++. It employs the ncurses C/C++ user interface library.

A start-up delay may occur. Compatible browsers include recent versions of Chrome, Edge and Firefox.

SHA1 Hash to PIN Reverter

This is a web app that provides a tool that can reverse a SHA1 PIN hash back to its associated PIN. The app is built with HTML, CSS and PHP.

A start-up delay may occur. Compatible browsers include recent versions of Chrome, Edge and Firefox.

Subject Notes

This is a web server application for collaboratively compiling notes about a subject. The app is developed with Python, Flask and PostgreSQL. It can authenticate any user with a Google account.

Compatible browsers include recent versions of Chrome, Edge and Firefox.

Matching Game

This is a one player game about revealing cards by matching. The game is built with HTML, CSS and JavaScript.

Browsers compatible with video demo page include recent versions of Chrome, Edge and Firefox.

Markov Chain Sim

This is a Markov-chain driven simulation of the positional states of hard disks contained by walls bounding a square space. This project is written in Python with the matplotlib and random libraries. It was developed in a Jupyter notebook.
