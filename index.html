A Portfolio of Exhibition Projects

by J. F. Padilla

Browsers compatible with video demo page include Chrome 81, Firefox 76 and Edge 81.

Disk Simulation

This is a 2D simulation of binary collisions among disks moving along straight paths. Separate threads concurrently process the animation and the physics. This project is written in C++17. It employs the SDL library for rendering graphics.

Traffic Simulation

This is a city traffic simulation employing a graph model. Separate threads concurrently process each vehicle, intersection and traffic light. This project is written in C++. It employs C++ interfaces of the OpenCV software library.

Linux Monitor

This is a Linux system monitor. Its output includes memory and CPU usage of the system's most intense software processes. This project is written in C++ 2017. It employs the ncurses C/C++ user interface library.

A start-up delay may occur. Compatible browsers include Chrome 81, Firefox 76 and Edge 81.

Subject Notes

This is a web server application for collaboratively compiling notes about a subject. The app is developed with Python, Flask and PostgreSQL. It can authenticate any user with a Google account.

Compatible browsers include Chrome 81, Firefox 76 and Edge 81.

Matching Game

This is a one player game about revealing cards by matching. The game is built with HTML, CSS and JavaScript.

Browsers compatible with video demo page include Chrome 81, Firefox 75 and Edge 81.

Markov Chain Sim

This is a Markov-chain driven simulation of the positional states of hard disks contained by walls bounding a square space. This project is written in Python with the matplotlib and random libraries. It was developed in a Jupyter notebook.
